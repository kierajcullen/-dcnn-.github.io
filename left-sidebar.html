Introduction

This document describes the architecture and implementation for an end-to-end 2D convolution acceleration system, targeting applications such as image filtering and deep convolutional neural networks.

General Information

FPGA

Nexys DDR board with Xilinx Artix-7 chip, 160 18-bit DSP slices (up to 16 GOPs throughput at 100 MHz), > 0.5 Mbytes of block RAM

Same

Output image same size as input image (f x f)

Full

Output image is larger than input image ((f+g) x (f+g))

Valid

Output image is smaller than input image ((f+g/2) x (f+g/2))
